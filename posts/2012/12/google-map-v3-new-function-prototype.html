Google Map v3 new function prototype

December 9 2012, 4:35 PM  by Mar Cejas

Google Map API has new function in getting the location data. One of this function is when getting the Latitude and Longitude which is now using the following:

geo.geocode({'address':Address},function(result,status){ if(status=='OK'){ var pos = result[0].geometry.location; console.log(pos.lat()); console.log(pos.lng()); } });

Did you notice the new function lat() and lng() rather than before using object Xa & Ya?

This update had effected around October 2012, I think??
