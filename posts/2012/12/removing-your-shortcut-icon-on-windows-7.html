Removing your shortcut icon on Windows 7

December 9 2012, 4:07 PM  by Mar Cejas

Actually, this is not removing the shortcut icon but rather we are making it transparent icon instead of the arrow icon. 

How we do it, is we need to access it on registry using regedit command on command prompt. Once we are on the registry, we need to find the following keys:

  • HKEY_LOCAL_MACHINE
  • SOFTWARE
  • Microsoft
  • Windows
  • CurrentVersion
  • Explorer
  • Shell Icons (This key is not created on default Windows 7 registry, so we need to create this manually by doing a right-click and create new key)

Once, we have created the Shell Icons key, we need then create a new String Value with the following Name: 29 and Data: %SystemRoot%\System32\imageres.dll,113

Logoff your computer and then see the cleaner shortcut icons on your desktop.

Note: Sometimes the shortcut icon turn black, I think problem hooking up from the icon library, not sure, maybe someone can comment why this is happening. So, another option is to create a transparent icon and hook them up as your new Data value.
